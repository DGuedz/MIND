# MIND Backend Economy Skill

**Layer:** Backend Optimization · Token Economy Compliance  
**Focus:** VSC Policy Alignment · Minimal Token Usage  
**Compliance:** Strict VSC Economy Mode  

## Core Principles

1. Zero decorative emojis - functional indicators only
2. Minimal formatting - essential content focus  
3. Clear calls-to-action - direct communication
4. Token optimization - reduce redundancy
5. Policy compliance - adhere to VSC guidelines

## Implementation Rules

- NO_EMOJIS: Remove all non-functional decorative elements
- MINIMAL_FORMATTING: Use only essential formatting for clarity  
- ESSENTIAL_CONTENT: Focus on crucial information only
- CLEAR_CTAS: Direct and unambiguous calls-to-action
- TOKEN_OPTIMIZED: Minimize repetition and decorative content

## Usage

This skill automatically applies VSC economy mode to all backend operations:
- API responses
- Log formatting  
- Documentation
- System messages
- Error reporting

## Quality Gates

- All output validated for token economy compliance
- Regular audits for policy adherence
- Performance metrics tracking token savings
- Continuous optimization based on VSC guidelines
