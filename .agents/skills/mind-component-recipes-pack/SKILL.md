# MIND Component Recipes Pack

## Objetivo

- Padronizar componentes para reuso
- Estilo premium dark metallic com acento Solana contido
- Saída com poucos tokens e sem dependências novas

## Regras

- Usar classes existentes do projeto quando possível
- Evitar componentes grandes com efeitos de blur
- Preferir grid/flex com `min-w-0` e `truncate`

## Saída

- 5 a 8 snippets TSX prontos
- Lista de tokens e classes base usadas
