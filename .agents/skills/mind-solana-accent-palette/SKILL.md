# MIND Solana Accent Palette

## Objetivo

- Base cinza como padrão (zinc)
- Acentos Solana apenas como detalhe (borda, shine, micro-gradiente)

## Regras

- Verde Solana: usar em estados positivos, highlights curtos
- Roxo Solana: usar em status/links e micro-contraste
- Evitar neon: sem sombras grandes e saturadas
- Garantir contraste mínimo para texto

## Saída

- Tokens de cor (hex + opacidades recomendadas)
- Receitas de gradiente (fundo, borda, shine)
- Checklist de acessibilidade
