# SKILL: PMF Validator

## Objective
Analyze a project idea and return structured Product-Market Fit insights.

## Input
- project_description: string

## Output
- pmf_score: number (0-100)
- strengths: array
- weaknesses: array
- suggestions: array

## Execution
1. Analyze clarity of problem
2. Identify target user
3. Evaluate differentiation
4. Suggest improvements

## Acceptance Criteria
- Output must be structured
- No generic answers
- Must highlight at least 1 real risk

## Evidence
- Input + Output logged
