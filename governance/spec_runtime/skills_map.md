# Skills Map (Spec-Driven Runtime)

Como infraestrutura institucional para o **Agentic GDP** na Solana, o MIND consolida seus dados através das seguintes rotinas de especificação e aprendizado contínuo:

## skill_compile_knowledge
Input: raw strategic text + **Colosseum Copilot structured data** (Market Intel from 5.400+ builder projects & Cypherpunk archives)
Output: updates to thesis, revenue, pitch, and execution docs

## skill_update_index
Input: current docs
Output: refreshed `mind_index.md`

## skill_file_learning
Input: successful outputs, decisions, **Deep Dive validations (Colosseum Hackathon/Archive signals)**, and **Verified Execution Receipts (Metaplex Mindprint + Turnkey TxHashes)**
Output: durable entries in core docs (not chat-only memory) and immutable audit trails.

## skill_lint_memory
Input: all spec docs
Output: contradictions, overclaims, missing evidence, open questions

## skill_assemble_context
Input: topic (optional)
Output: Tier 1/Tier 2 context package for next generation step

## skill_promote_to_spec
Input: repeated pattern from runs
Output: codified spec section + checklist item + metric target
